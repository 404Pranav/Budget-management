# LuxeBudget - Project Structure

This project has been reorganized for better maintainability and clarity.

## Directory Structure
- `assets/images/` - Contains all visual media and icons used in the application.
- `css/` - Contains all stylesheets (`style.css`).
- `js/` - Contains all JavaScript logic and application state management (`app.js`).
- `index.html` - The main dashboard entry point.
- `login.html` - The authentication page.

## How to use
The paths in the HTML files have been updated to reflect this new structure. No further changes are required for the application to function correctly.
